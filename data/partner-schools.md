# A+ Tutoring — Partner Schools Reference

This file is the **canonical source of truth** for which charter schools A+ Tutoring has provided services to in the last 2 years. The YAML frontmatter above is the machine-readable list; the prose below is for human reference.

## Overview

A+ Tutoring has provided services to **27 charter schools** in the last 2 years (as of 2026-05-22). All 27 are currently ACTIVE partnerships. None have been marked FORMER yet — when a partnership ends, change its status field but do not delete the entry, because case studies and blog posts may reference former partners and the URL slug needs to stay reserved.

**Of the 27 active partnerships, 6 have dedicated landing pages** on wetutorathome.com:

- iLEAD Exploration → `/ilead-exploration`
- Sky Mountain → `/sky-mountain`
- Sage Oak → `/sage-oak`
- Heartwood Charter School → `/heartwood-charter-school`
- Blue Ridge Academy → `/blue-ridge-academy`
- Granite Mountain → `/granite-mountain`

The other 21 are operational partnerships without dedicated marketing pages. They may receive landing pages in the future as the partnership deepens, at which point the `has_landing_page` field gets updated to `true` and the `landing_page_url` field is filled in.

## How URL slugs work

The `url_slug` field is used to construct case-study URLs on the Case Study blog:

```
blog.wetutorathome.com/case-study/{student_pseudonym}-{school_url_slug}
```

For example, Gabriela (pseudonym) at iLEAD Exploration publishes to:

```
blog.wetutorathome.com/case-study/gabriela-ilead
```

The slug pattern is chosen for SEO clustering — every iLEAD case study URL contains `-ilead`, which over time builds topical authority for iLEAD-related parent searches.

**Once a case study is published for a school, that school's `url_slug` is frozen.** Changing it would break live URLs and require 301 redirects. New URL slugs can be added freely; existing ones must not be changed.

## How to add a new school

When A+ Tutoring begins providing services to a new charter partnership:

1. Add the new partnership in HubSpot Companies (this is the operational system of record)
2. Add a new entry to the `schools:` list in the YAML frontmatter of this file, following the same schema
3. Commit the change to the `aplus-marketing-skills` repo in the same PR/commit as any related work
4. If the new school will get a dedicated wetutorathome.com landing page, update `has_landing_page` to `true` and fill in `landing_page_url` once the page is live

## How to mark a school as former

When a charter partnership ends:

1. Change the school's `status` from `ACTIVE` to `FORMER`
2. Add a `partnership_ended` field with the date the partnership ended
3. Do not delete the entry — historical case studies and blog posts may reference the former partnership
4. Do not change the `url_slug` — published URLs depend on it

## Naming conventions baked in

A few editorial decisions captured here so future writers don't redebate them:

- **"Blue Ridge Academy" is two words** in published content, even though some internal systems spell it "Blueridge" as one word. Use the canonical two-word form in all marketing content.
- **"iLEAD" is the short form**, not "iLead" or "ILEAD." The capitalization pattern is iLEAD (lowercase i, capital LEAD). Brand-check should flag any variant.
- **"EPIC" stays in all caps** in published content (acronym for the network's full name).
- **"BEST" stays in all caps** for the same reason.
- **"Heartwood Charter School" can be shortened to "Heartwood"** inline, but the first reference in any piece should use the full canonical name.
- The same shortening convention applies to all schools: full canonical name on first reference, `short_name` on subsequent references.

## Spelling and disambiguation notes

- **"Visions in Education"** is the full name; do not abbreviate to "Visions" without context, as it's ambiguous (other CA charters use similar names)
- **"Compass Charter Schools"** is plural ("Schools"), reflecting the network structure
- **"Horizon Charter Schools"** is also plural for the same reason
- **"EPIC Charter Schools"** is also plural
- **"Sage Oak Charter Schools"** is also plural — single locations are uncommon in CA charter networks
- **"Springs Charter Schools"** is also plural

## What is NOT in this file

This file contains charter school partnerships ONLY. It does NOT contain:

- Private school partnerships (none currently)
